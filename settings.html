General
General settings for the site.
Navigation Bar
Change between a sidebar and a horizontal bar.

Instantly Launch Game
When you click on a game it will instantly go to it instead of viewing the splash screen of information.

Show Game Thumbnail
This hides the game thumbnails and makes it more compact. This also improves loading times and reduces network usage.

FPS Count
Toggle the frames per second while playing a game. Count will appear as average/minimum FPS.


Personalization
Personalize the site to your likings.
Mode
Change the website's appearance by a dark and a light contrast.

Theme Color
This changes the website's main color that you might see when using light or dark themed mode.

Tab Cloak
This changes the website icon and title to help hide this among your tabs. Note: This may not be working correctly.

Show Game Thumbnail Text
Toggle whether to show the text on game thumbnails. When this setting is off it only appears on hover.


Other
Other general site settings. Some of this might be useful others not.
Hot Keys
Hot keys allow you to navigate the site much easier. You can view them using Shift + /

Open Sidebar by Default
This will keep the sidebar open and only close when you play a game or if you close it.

Always on Top Navbar
For the horizontal navbar, this will keep it at the top of the screen even when scrolling.
